Java Schulung für ...

—  Privatpersonen

  • Sie möchten Programmieren lernen?
  • Nur aus Büchern lernen ist nicht Ihr Ding?
  • Sie haben keine ganze Woche Zeit für einen Blockkurs?

—  Firmen

  • Ihre Mitarbeiter sollen Programmieren lernen?
  • Learning-on-the-Job bringt zuwenig?
  • Sie wünschen effiziente Wissensvermittlung innerhalb kurzer Zeit?
  • Sie können es sich nicht leisten, Mitarbeiter tagelang in einen Kurs zu schicken?

Unser Konzept

Wir bieten JAVA Ausbildung an für:
  • Privatpersonen
    • als Abendkurse bei einer unserer Partnerschulen in Bern, Zürich und Basel
  • Firmen
    • als halbtägige Inhouskurse

Der Vorteil ist, dass Sie sich als Kursteilnehmer mit der Thematik über längere Zeit beschäftigen können und somit einen besseren Lernerfolg haben.
