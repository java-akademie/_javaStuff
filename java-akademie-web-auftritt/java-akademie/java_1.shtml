Java I

Einführung in die Java Programmiersprache

Zielsetzung

Java-Programmierer sind gesuchte Fachleute am Informatikmarkt. Die Programmiersprache Java erfreut sich einer grossen Beliebtheit sowie einer immer breiter werdenden Unterstützung auf allen Ebenen der Informatik. Java Programme sind Plattform unabhängig und portabel. Der Traum der Softwareentwickler "write once - run everywhere" wird durch Java zur Wirklichkeit. Die ebenfalls in der Sprache eingebaute Unterstützung von Netzwerk-Computing und Threading (Nebenläufigkeit) sowie seine strikten Sicherheitsmechanismen haben Java zur Sprache der Wahl für verteilte Applikationen im Internet und Intranet werden lassen.

Voraussetzung

Gute Informatik-Grundkenntnisse, sowie der Wunsch, sich solide Grundlagen der Programmierung anzueignen.

Methoden

Gruppenunterricht mit Lehrer und PC-Praxis.
Pro Woche ist mit 6 - 8 Stunden Hausaufgaben und Übungen zu rechnen.

Kursdurchführung

Es wird grosser Wert auf eine ganzheitliche Programmierausbildung und auf praktische Übungen gelegt — sowohl Einzelübungen als auch eine Fallstudie die Sie begleiten wird (eine Adressverwaltung).

Während dieser Fallstudie werden Sie alle Java-Komponenten die für die Programmierung nötig sind kennen lernen. Zu Beginn werden Sie Ihre Programme mit einem Texteditor erstellen und über die Kommandozeilenkonsole kompilieren und testen. Später werden Sie ECLIPSE zur Entwicklung verwenden.

Am Ende dieses Kurses sind Sie in der Lage einfache bis mittelschwere strukturierte Programme zu schreiben.
