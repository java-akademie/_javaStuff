Java II

Java für Fortgeschrittene

Zielsetzung

Sie kennen nun Java! Sie wissen alles über die Sprache ... wie man Code schreibt, die Syntax, OO ist Ihnen geläufig. Sie haben schon etwas Javapraxis.
Der Java II Kurs vertieft und erweitert Ihre bisherigen Kenntnisse und befähigt Sie, grössere Java Projekte professionell durchzuführen.

Voraussetzung

Java Kenntnisse wie Sie sie in Java I erworben haben oder gleichwertige anderweitig erworbene Kenntnisse.

Methoden

Gruppenunterricht mit Lehrer und PC-Praxis.
Pro Woche ist mit 6 - 8 Stunden Hausaufgaben und Übungen zu rechnen.

Kursdurchführung

Es wird grosser Wert auf eine ganzheitliche Programmierausbildung und auf praktische Übungen gelegt — sowohl Einzelübungen als auch Fallstudien die Sie begleiten werden.

Als integrierte Entwicklungsumgebung (IDE) werden Sie ECLIPSE verwenden. Als Datenbank werden Sie H2 und/oder MySQL kennen lernen.

Am Ende dieses Kurses sind Sie in der Lage anspruchsvolle Java Anwendungen selbständig zu entwickeln.
