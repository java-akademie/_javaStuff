JEE BCD

Java EE Business Component Developer

Zielsetzung

Dieses Seminar richtet sich an Java-Entwickler, die sich darauf spezialisieren wollen, EJBs (Enterprise Java Beans) und JPA (Java Persistence API) für die Entwicklung von robusten mehrschichtigen verteilten Applikationen zu verwenden.

Voraussetzung

Java Kenntnisse wie Sie sie in JAVA I und Java II und JEE WCD erworben haben oder gleichwertige anderweitig erworbene Kenntnisse.

Methoden

Gruppenunterricht mit Lehrer und PC-Praxis.
Pro Woche ist mit 6 - 8 Stunden Hausaufgaben und Übungen zu rechnen.

Kursdurchführung

Es wird grosser Wert auf eine ganzheitliche Programmierausbildung und auf praktische Übungen gelegt — sowohl Einzelübungen als auch Fallstudien die Sie begleiten werden.

Als integrierte Entwicklungsumgebung (IDE) werden Sie ECLIPSE, als Datenbank MySQL und als Applikationsserver JBOSS verwenden.

Dauer 12 Wochen à 4 Lektionen = 48 Lektionen
Daten Mittwoch oder Freitag 17:45 - 21:00 eventuell 6 Samstage
