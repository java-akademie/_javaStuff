Zielsetzung

Dieses Seminar richtet sich an Java-Entwickler die sich darauf spezialisieren wollen, JSP (Java Server Pages) und die Java Servlet Technologie für die Entwicklung von Web Anwendungen sowie EJB's (Enterprise Java Beans) für die Entwicklung von robusten mehrschichtigen verteilten Applikationen zu verwenden.

Voraussetzung

Java Kenntnisse wie Sie sie in JAVA I und JAVA II erworben haben oder gleichwertige anderweitig erworbene Kenntnisse.

Methoden

Gruppenunterricht mit Lehrer und PC-Praxis.
Pro Woche ist mit 6 - 8 Stunden Hausaufgaben und Übungen zu rechnen.

Kursdurchführung

Es wird grosser Wert auf eine ganzheitliche Programmierausbildung und auf praktische Übungen gelegt — sowohl Einzelübungen als auch Fallstudien die Sie begleiten werden.

Als integrierte Entwicklungsumgebung (IDE) werden Sie ECLIPSE, als Datenbank MySQL und als Applikationsserver JBOSS verwenden.
